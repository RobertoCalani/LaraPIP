Larapip

Laravel + Docker — interactive stack builder, one command.

Larapip is a Docker configurator. It generates docker-compose.yml, Dockerfile, and .env for you. After setup, you can use plain docker compose or the included launcher.

$ curl -fsSL https://robertocalani.github.io/LaraPIP/installer.sh | bash
GitHub

Features

Zero Config

Answer 6 prompts, get a fully Dockerized Laravel stack. No manual setup.

FrankenPHP

HTTP/2, Octane, worker mode — all built into the PHP server.

Auto SSL

Caddy auto-generates local certificates. Dev HTTPS in one click.

Portable

Docker-only. Consistent environment on Linux, macOS, or Windows (WSL2).

Launcher

./larapip.sh optional helper for containers, DB shell, backups, and more.

DB Volume

All engines store data in a single database volume. Inspect anytime.

Usage Commands

Using larapip.sh is optional. It's a convenience wrapper around docker compose — you can use Docker directly (see the "Vanilla Docker" tab).
Larapip CLI commands
CommandWhat it does
./larapip.sh upStart all containers
./larapip.sh downStop all containers
./larapip.sh db:shellConnect to database CLI (auto-detects engine)
./larapip.sh db:volumeShow database volume info and size
./larapip.sh db:backupDump database to backups/
./larapip.sh db:restoreRestore database from dump
./larapip.sh artisan <cmd>Run Artisan commands in container
./larapip.sh logs [svc]Tail logs (app, database, node, etc.)
./artisanShorthand for Artisan (bridge script)
./composerShorthand for Composer (bridge script)

Frequently Asked Questions

Need PHP installed?
No. Everything runs in Docker.
Production ready?
Designed for dev. Review SSL before production.
Caddy & SSL for production?
Edit config_docker/frankenphp/Caddyfile:
• Change localhost → your domain
• tls your@email.com (auto Let's Encrypt) or tls /path/cert.pem /path/key.pem (manual)
• Add header / Strict-Transport-Security max-age=31536000; for HSTS
• Remove auto_https off to let Caddy manage TLS

SSL — Option A (auto): set domain in Caddyfile → Caddy gets a Let's Encrypt cert automatically (needs port 80).
SSL — Option B (manual): put .pem and .key in config_docker/ssl/, run installer with SSL_MODE=manual.
FrankenPHP tuning?
In the Caddyfile, add inside php_server:
php_fastcgi max_execution_time 30 max_requests 500

In .env:
• APP_ENV=production
• APP_DEBUG=false

For Octane:
php artisan octane:start --server=frankenphp --host=0.0.0.0 --port=80
.env changes for production?
APP_ENV=production
APP_DEBUG=false
APP_URL=https://yourdomain.com ← required for absolute URLs (routes, emails, signed URLs)
APP_KEY — generate with php artisan key:generate
LOG_LEVEL=warning
SESSION_DRIVER — e.g. redis
CACHE_DRIVER — e.g. redis
QUEUE_CONNECTION — e.g. database or redis
Scale queue workers?
In docker-compose.yml, duplicate the jobs service with different container_name and:
command: php artisan queue:work --tries=3 --timeout=90

For Docker Swarm, add deploy.replicas: N to each job service.
Change DB password?
Edit .env, restart with docker compose up -d.
DB volume name?
Always <project>_database. Check with docker volume ls.
Need larapip.sh?
No — it's optional. Use raw docker compose. See the Vanilla Docker tab.
Laravel 11 safe?
EOL since Mar 12, 2026. Use 12 or 13.
Windows?
Yes, via WSL2.
